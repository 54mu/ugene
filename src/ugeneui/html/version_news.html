Changes in UGENE 1.28.1

  • Tweaking of the Sanger reads mapping feature, based on feedback received
  • Fix of a critical issue of searching in amino acid translations of a nucleotide sequence
  • Other minor bug fixes and improvements

Any feedback and suggestions about the UGENE Sanger Reads Editor are welcome!

Изменения в UGENE 1.28.1

  • доработка алгоритма выравнивания прочтений по Сэнгеру с учетом отзывов пользователей,
  • исправление критической ошибки поиска по аминокислотным трансляциям нуклеотидной последовательности,
  • исправление мелких ошибок и другие улучшения.

Мы ждем ваших отзывов по поводу редактора данных секвенирования по Сэнгеру!
