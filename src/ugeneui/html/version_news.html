Changes in UGENE 1.31

The new version introduces a lot of major features for NGS data analysis: metagenomics data classification, transcriptomics, de novo assembly, etc.
See details on the UGENE web site.

Изменения в UGENE 1.31

В новой версии значительно расширена инфраструктура для анализа данных высокопроизводительного секвенирования (NGS) в таких областях, как таксономическая классификация метагеномных данных, транскриптомика, de novo сборка и др.
Подробности см. на сайте UGENE.
